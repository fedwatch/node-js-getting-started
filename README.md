# 前端价值面试指南
StartBit，前端价值面试指南(Front end value interview guide)是由沪上知名的前端招聘团队在多年IT前端招聘流程过程中总结出来的正确有效的实践方法和基于FVIG价值评估体系研究出来综合成果，旨在帮助中小企业主更好更快的低成本找到合适的前端工程师和高级工作人员。
